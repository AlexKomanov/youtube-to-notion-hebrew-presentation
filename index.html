שלב 1 — אפשרות א'

התקנת MCP — דרך CLI

פקודה אחת בטרמינל, Claude Code מטפל בשאר

  • 🔗
    MCP — Model Context Protocol — מאפשר ל-Claude לדבר ישירות עם שירותים חיצוניים כמו Notion
  • ⚙
    מתאים למי שעובד עם Claude Code CLI ורוצה שליטה מלאה
  • 🔑
    Claude Code יפתח דפדפן לאישור OAuth — ללא API Key ידני
  • ✅
    לאחר ההרצה, Notion זמין מיד לכל הפרויקטים
Terminal 
$ claude mcp add --transport http notion https://mcp.notion.com/mcp

שלב 2

התקנת yt-dlp

הכלי שמאפשר לשלוף מטא-דאטה מסרטוני YouTube

  • 💾
    yt-dlp — כלי command-line בקוד פתוח לשליפת מידע מ-YouTube ועוד
  • 📋
    מאפשר לשלוף כותרת, ערוץ, משך, תמונה ממוזערת — ללא צורך ב-YouTube API
  • 🚀
    Claude Code מריץ אותו ברקע — המשתמש לא רואה כלום, הכל קורה אוטומטית
  • 🤓
    חינמי, מהיר, ועובד עם כל URL של YouTube — כולל Shorts
macOS 
$ brew install yt-dlp
Windows 
# via Windows Package Manager
> winget install yt-dlp

# via Python (cross-platform)
> pip install yt-dlp

youtube-videos-to-notion-db

במקום לשמור ידנית — פשוט להדביק URL ו-Claude עושה הכל

🤔
למה להשתמש בסקיל? — בלי הסקיל, כל שמירת סרטון דורשת: פתיחת Notion, מציאת הדאטאבייס, הזנת שם + ערוץ + משך + URL ידנית, בדיקה שלא כפלתם. עם הסקיל — URL אחד בצ'אט, הכל נעשה אוטומטית תוך שניות.
1
מזהה URL — מחלץ את מזהה הסרטון מקישור YouTube (כולל Shorts, playlists)
2
מושך מטא-דאטה — מריץ yt-dlp לשליפת כותרת, ערוץ, משך, תמונה ממוזערת
3
בודק כפילויות — מחפש ב-Notion אם הסרטון כבר קיים ברשימה
4
מקצה Bucket — מנתח את הכותרת והתוכן, מחליט לאיזה קטגוריה שייך הסרטון
5
מוסיף ל-Notion — יוצר רשומה חדשה עם כל השדות בפעולה אחת דרך MCP
⭐
